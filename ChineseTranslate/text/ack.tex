\section{致谢 (Acknowledgements)}

本工作的大部分内容在很大程度上基于他人的工作。Web3 基金会研究团队，特别是 Alistair Stewart 和 Jeff Burdges，负责 \textsc{Elves}，即 Polkadot 的安全机制，使 \Jam 的核心内计算成为可能。同一团队还负责 Sassafras、\textsc{Grandpa} 和 \textsc{Beefy}。

Safrole 是 Sassafras 的适度简化版本，经过 Davide Galassi 和 Alistair Stewart 的仔细审查后完成。

原始的 CoreJam \textsc{rfc} 在 Bastian Köcher 和 Robert Habermeier 的审查下得到了改进，该提案的大部分关键要素已被纳入本工作。

\textsc{pvm} 是 Jan Bujak 开发的部分简化的 \emph{PolkaVM} 软件原型的形式化。Cyrill Leutwiler 对本工作报告的 \textsc{pvm} 实证分析做出了贡献。

\emph{PolkaJam} 团队，特别是 Arkadiy Paronyan、Emeric Chevalier 和 Dave Emett，在 \Jam 协议底层方面的设计中发挥了重要作用，尤其是涉及 Merklization 和 \textsc{i/o} 的部分。

自发表以来，众多仓库贡献者帮助纠正了错误。感谢所有人。

当然，也要感谢出色的 Lemon Jelly，又名 Fred Deakin 和 Nick Franglen，献上了有史以来最美丽的三张专辑，第一张专辑的封面艺术启发了本文的背景设计。
